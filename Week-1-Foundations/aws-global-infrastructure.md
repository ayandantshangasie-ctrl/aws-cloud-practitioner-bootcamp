Uses multiple Availability Zones (AZs) to keep applications running even if one data centre has a problem.If one AZ fails because of power or network issues, another AZ can continue working, so the website or application does not go offline.
It helps applications run faster, stay available, and keep data safe by using regions, availability zones, and networks across different countries,has data centres all around the world so people and businesses can use cloud services from locations closer to them.
